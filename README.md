# AI-vibe-coded-binary-QR-code-scanner
QR code scanner for scanning data out of binary QR codes
delete the packages.json and delete the package-lock.json and then refer to the readme.txt file for more information on what needs to be done dependencies etc
```bash
sudo apt install nodejs npm -y
npm init -y
npm install express
```
to downloads the source code you can download the zip form github or use git
```bash
git clone https://github.com/soniceggman419-ship-it/AI-vibe-coded-binary-QR-code-scanner.git
```
